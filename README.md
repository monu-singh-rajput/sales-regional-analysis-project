# Sales Analytics Dashboard (2014–2018)

## Overview

End-to-end Sales Analytics project analyzing 64K+ records, including data cleaning, feature engineering, database integration, and interactive dashboard development to generate business insights.

## Tools Used

* Python (Pandas, NumPy)
* MySQL
* Power BI (Dashboard – In Progress)

## Data Pipeline

Excel → Python (Data Cleaning & Feature Engineering) → MySQL → Power BI

## Key Features

* Analyzed 64,000+ sales records
* Profit margin and cost analysis
* High-value customer identification (Top 10%)
* Regional and channel performance analysis
* Seasonal trend detection

## Business Insights

* Revenue is highly concentrated among a small group of customers and products
* The wholesale channel contributes the majority of total revenue
* The West region is the highest revenue contributor
* Clear seasonal demand pattern with peak sales between May and August

## Project Structure

* **Data/** – Raw dataset
* **Python/** – Data cleaning and feature engineering
* **SQL/** – Database creation and analysis queries
**Database:** sales_analytics_db  
**Table:** sales_data
